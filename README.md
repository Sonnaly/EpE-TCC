# Elas Por Elas  
É um site básico, com propósito informátivo, para auxiliar mulheres visitas de violência.

# Funcionalidades  
* Painel de dados;  
* Adicionar Comentários;
* Deletar Comentários;
* Persistencia de dados através de LocalStorage.

# Tecnologias Utilizadas  
* HTML5;
* CSS3/BOOTSTRAP4;
* JavaScript;
* Node.js e Scocket.io.

# Melhorias Futuras  
* Tornar os comentários em um chat de interação entre as usuárias;  
* Atualizar os gráficos automaticamente através de dados governamentais;
* Transformar em uma aplicação mobile.
